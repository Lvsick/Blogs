# 数组遍历

# forEach() 方法对数组的每一个元素执行一次回调函数
# 基础语法：array.forEach(function(currentValue,index,arr),thisValue)
# 参数：
  • currentValue,数组当前元素
  • index(可选)，索引
  • arr(数组对象本身)
# 注意:
  • 总是返回undifined
  • 不能修改数组本身，即 forEach 不会直接改变调用它的对象，但是那个对象可能会被 callback 函数改变，主要用于遍历出数组的每一项元素
# map()对数组中的每个元素进行处理，返回新的数组
# 基础语法：let new_array = arr.map(function(currentValue, index, arr), thisArg)
# 参数：同上
# 注意：回调函数中不能忘了return
# filter() 过滤原始数组，返回新数组
# 基础语法：let new_array = arr.filter(function(currentValue, index, arr), thisArg)
# 参数：同上
# 注意：回调函数中不能忘了return
