# Reasons

# 自己突然放松。

# 不知道什么原因就开始了无所事事了，完全提不起学习的劲头，对自己要求变得更加低下，可能跟去第一个实习公司有关，第一个实习公司面试的是前端，进去让我们学习flutter，然后学习了一段时间之后就不了了事，大概就从那时候开始放松了，什么都不想干，只想躺着。

# 果断辞了实习，找到了下一家，现在就是入职的第一天，没事干，就写个东西说说吧。

# 还有一大部分原因应该跟我出来实习有关。出来实习，租个房间，买这买那，折腾来折腾去，然后就变成这样了。过年回来去就没有认真看过相关的知识，只有前几天把TS给过了一遍，基本上也没啥问题。

# Now

# 来到新的实习单位，是一个小公司，主要是离我自己住的地方比较近，真的很近，而且重庆的实习真的太难找了，跟成都和杭州这种互联网大城市根本没法比，我实在不想折腾了，所以面试过后也没去面试其他单位，因为我不想折腾了，如果要去其他城市，自己的那一堆破烂不知道要去放到哪里，就先这样干着吧，看看后面会有什么惊喜。

# Change

# 首先，保持天天进步。

# 其次，保持blog的更新，督促自己学习。

# 最后，加强数据结构与算法的学习。
